## âš¡ Quick Commands

### First Time Setup
```bash
# 1. Create database
mysql -u root -p
CREATE DATABASE qatar_tourism;
source database/schema.sql;

# 2. Backend setup
cd backend
npm install
copy .env.example .env
# Edit .env with your credentials
npm run dev

# 3. Frontend setup (new terminal)
cd frontend
npm install
npm start
```

### Daily Use
```bash
# Start backend
cd backend
npm run dev

# Start frontend (new terminal)
cd frontend
npm start
```
